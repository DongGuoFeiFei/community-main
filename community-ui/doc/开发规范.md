## 层级设置建议

### 推荐的 Z-Index 规范

为了避免 z-index 混乱，建议按功能划分层级范围：

```scss
// Z-Index 层级规范
$z-index-background: -1; // 背景装饰元素
$z-index-content: 1; // 普通内容
$z-index-dropdown: 100; // 下拉菜单
$z-index-sticky: 200; // 固定定位元素
$z-index-modal-backdrop: 300; // 模态框背景
$z-index-live2d: 400; // Live2D 模型
$z-index-live2d-control: 500; // Live2D 控制面板
$z-index-back-to-top: 600; // 回到顶部按钮
$z-index-modal: 1000; // 模态框
$z-index-notification: 2000; // 通知提示
$z-index-tooltip: 3000; // 工具提示
```
